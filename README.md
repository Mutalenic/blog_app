## Blog App: validations, model specs, and n+1 problems

In this project, I have added validations to my models, created specs for them, and found and fixed N+1 queries problems.
## Built with
- Ruby
- Rubocop
- Stylelint
- RoR

## SetUp
To get a local copy up and running follow this steps
- git clone `https://github.com/Mutalenic/blog_app.git`
- Run `bundle install`

# Run the project
`rails s`

### Run linters
```
rubocop -a

### Tests
```
rspec spec
```
